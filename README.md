Tarmo - Tampere Mobilemap

### Development

Use Node>=v16.

1. Copy .env.sample as .env and fill your NLS Api key.
2. Install dependencies:
   ```shell
   yarn
   ```
3. Run the development server in localhost:3000:
   ```shell
   yarn run start
   ```
